Making Web App Fullscreen

Jan 2, 2017 Web Fundamental

TO BE CONTINUED…

Google developer has a very good post about how to make a web app fullscreen. Long story short, only iOS Safari doesn’t have a fullscreen API, and we can use element.requestFullscreen() (if prefixed) in all other modern browsers.

We can launch a fullscreen web app from home screen on iOS with the help of meta tag <meta name="apple-mobile-web-app-capable" content="yes">. One annoying thing is that because after iOS6, all home screen web app run in a sandbox other than Safari, the session cannot be kept. See this Stack Overflow post for solutions.
